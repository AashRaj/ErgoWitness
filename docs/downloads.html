Installation Steps

Windows

  1. Download and unzip Ergo Witness
  2. Run the .exe file in the extracted folder
  3. Pick your quality settings
  4. Type in the address of your ELK server
  5. Type in the IP address's that are of interest to you
  6. Make sure that you hit all the needed "Apply" buttons, and then press "Done"
  7. Press play, and you are done!
